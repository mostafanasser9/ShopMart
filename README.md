# ShopMart

Next.js + Tailwind e-commerce demo application.

## Summary

ShopMart is a small storefront demo using the Next.js App Router, TypeScript and Tailwind CSS.

## Local development

Install dependencies:

```bash
npm install
```

Run the dev server:

```bash
npm run dev
```

Open http://localhost:3000

Pull requests welcome. For large changes, open an issue first to discuss scope.
